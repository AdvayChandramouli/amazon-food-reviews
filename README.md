# Amazon Food Reviews - Sentiment Analysis

## Overview
This project analyzes product review sentiment using Python and NLP to classify reviews as positive, negative, or neutral.

**CS4375.001 - Group 20:** Advay Chandramouli, Alan Roybal, Ubaid Mohammed

## Methodology
- Data preprocessing and text cleaning
- Lexicon-based sentiment scoring with VADER and TextBlob
- Baseline supervised modeling with Naive Bayes
- Evaluation using standard classification metrics
- Basic visualizations for sentiment distribution and model performance

## Setup
- Load the data from [this dataset](https://www.kaggle.com/datasets/snap/amazon-fine-food-reviews/data).
- Modify `DATASET_CSV_PATH` to match your local directory structure.
- Install requirements by running the following command: 
    - ```pip install -r requirements.txt```
